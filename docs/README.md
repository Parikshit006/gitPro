# GitPro Documentation

## Structure

- **ADR**: Architecture Decision Records (ADRs). Formal documentation of significant architectural choices and trade-offs.
- **API**: API specifications and integration contracts.
- **Diagrams**: System architecture, sequence, and entity-relationship diagrams.
- **Decisions**: Non-architectural technical decisions, product decisions, and RFCs.

## Future Documentation

As the project evolves, this folder will also host deployment guides, environment configuration references, and operational runbooks.
